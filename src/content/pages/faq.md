## How can I save reconstruction results?

There are two ways:

1. Use `File → Save SWC` to save the results.
2. In the 3D window, select a reconstruction, right click, and choose `Save As`.

The reconstruction will be saved as an SWC file.

## Can I trace a multiple-channel image?

Yes. The target channel needs to be extracted first by `Tools → Process → Extract Channel`.

## What image file formats does neuTube support?

Supported formats include:

- Zeiss LSM format (`.lsm`)
- Limited TIFF format (`.tif` or `.tiff`)
  neuTube supports less than 2GB uncompressed or LZW-compressed classic TIFF files.
- Uncompressed Vaa3D raw format (`.v3draw`)
  This is the recommended format for tracing large image stacks and can be generated by Vaa3D or Fiji.
