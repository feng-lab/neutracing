## Getting started

### Installation

There is no need to install neuTube. All you need to do is decompress the downloaded zip file. To uninstall, simply delete the files.

### Launch the program

**Executables**

- Mac: `neuTube.app`
- Windows: `neuTube/neuTube.exe`
- Linux: `neuTube/neuTube`
